# Chunguscoin

Chunguscoin is an meme based, opensource, decentralized, peer-to-peer digital currency with a multipurpose design.

# Folder layout
```
./build -> output
./depends -> needed libs
./doc -> docs
./src -> code
./src/cli -> cli tool
./src/wallet -> QT Wallet
./src/komodo -> smart chain source code **not needed**
```
